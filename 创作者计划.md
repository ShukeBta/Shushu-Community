# 鼠鼠创作者计划

> 鼠鼠交易平台面向全网内容创作者推出「鼠鼠创作者计划」，通过视频推广平台服务获取丰厚奖励。

## 参与条件

| 条件 | 说明 |
|------|------|
| 平台 | 抖音、快手、B站、小红书、微博均可 |
| 粉丝量 | **无要求** |
| 内容方向 | 三角洲行动、游戏账号租赁、代练等 |
| 注册要求 | 鼠鼠交易平台注册 + 实名认证 |

## 参与流程

1. 注册 PD 租号平台 → 实名认证
2. 联系客服 → 提交创作者计划申请
3. 获取专属推广素材 + 邀请码
4. 发布视频（需包含平台推荐内容）
5. 提交视频链接给客服
6. 审核通过后按播放量结算
7. 奖励发放至余额，可随时提现

## 创作规范

✅ 真实体验分享、口播或文字提及平台、简介放邀请码

❌ 虚假宣传、刷量作弊、违规内容、盗用素材

## 结算规则

- 结算周期：视频发布 **7 天后** 统计
- 最低提现：50 元
- 到账方式：平台余额 → 支付宝/微信/银行卡
- 审核时间：1-3 个工作日

> 📩 报名：联系平台客服，发送「创作者计划报名」
